PHP 無法載入模組

前言

有時候建置 PHP 開發環境，
可能會依賴其他第三方的軟體庫
然後會發生一些見鬼的事情。

狀況描述

當安裝完畢之後通常會執行測試指令

sudo php7.4-fpm -t

這時候就會看各種模組載入的警告

解決方式

原因就是有些模組有相依關係，
所以要先載入被依賴的模組。

設定檔案要做調整，
可參考上圖左邊的檔案目錄 /etc/php/7.4/fpm/conf.d，
要額外加入 mysqlnd 和 pdo 並盡量將順序放在前面，
才能正確載入 mysqli 和 pdo_mysql。
